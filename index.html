¿Qué es Drobec-app?

Drobec-app es una aplicación móvil para crear páginas html estáticas basadas en drobec donde el usuario no necesita usar código . La app está diseñada para editar la página gráficamente sin acceder al código fuente de la misma.

La descarga y uso de la app para Android es gratis, pero cuenta con la compra de una licencia PRO para algunas funciones.
